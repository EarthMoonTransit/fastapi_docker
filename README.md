### FastApi and Docker 
Склонируйте репозиторий:

```
https://github.com/EarthMoonTransit/fastapi_docker.git 
```
Cоздайте виртуальное окружение:
```
python -m venv venv
```
Активируйте виртуальное окружение:
```
source venv/bin/activate
```
Запустите контейнер приложения docker
```
docker-compose up
```
